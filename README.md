# RAYLIB - biblioteca intuitiva para desenvolvimento de jogos e gráficos 3D em C

Avaliação do luiz: biblioteca com boníssimas abstrações adjunta do poder e
simplicidade funcional providas pela linguagem C. Se queres desenvolver
gráficos, jogos ou simulações físicas rapidamente (semelhantemente como fazes
no webGL em navegadores), raylib é a opção!
